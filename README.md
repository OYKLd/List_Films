# List_Films

## Description
Application web permettant de créer et gérer une liste de films à regarder, avec possibilité de marquer les films comme vus ou à voir.  
Ce projet est le **vingt-huitième** du défi personnel **100 projets en 2026**.

---

## Objectifs du projet
- Implémenter un CRUD simple
- Structurer une liste d’objets (films)
- Gérer des statuts dynamiques (vu / à voir)
- Mettre en place des filtres
- Améliorer l’organisation d’une interface en cartes

---

## Plateforme
- Web (navigateur)

---

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage

---

## Fonctionnalités
- Ajouter un film :
  - Titre
  - Genre
  - Année (optionnel)
- Marquer un film comme **vu**
- Marquer un film comme **à voir**
- Supprimer un film
- Filtrer les films :
  - Tous
  - Vu
  - À voir
- Sauvegarde locale des données

---

## Design & UX
- Films affichés sous forme de cartes
- Boutons de statut clairs (vu / à voir)
- Filtres visibles en haut de l’interface
- Design inspiré des plateformes cinéma
- Responsive (mobile et desktop)
